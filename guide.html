Welcome to Black Stars CRM

Everything you need to run the club: members, attendance, invoices, rentals, sales, coach commissions — all in one place, all offline. This guide walks through every daily task in order.

What this app does

Black Stars CRM is a complete management system for the sports club. It runs entirely in your browser — no internet needed once it's loaded. Your data stays on this computer.

👥
Members & Coaches
Register, renew, freeze, edit. Multi-sport support per member.
📄
Auto Invoices
Every member, renewal, sale, and rental gets an invoice automatically.
✓
Daily Attendance
Mark Y/N per student per day per sport. Filter, export, and report.
🛒
POS & Stock
Sell uniforms, gloves, drinks. Stock auto-decreases on sale.
🏟
Court Rentals
Football court, boxing room, swimming pool — book and invoice.
💰
Coach Commission
Per-coach % calculated automatically from active-member revenue.

Register a new member

  1. Go to Members in the sidebar.
  2. Click the red + Add Member button (top right).
  3. Fill in the form:
    • Name — required. Arabic name is optional.
    • Phone / Email / QID / Birthdate — optional but useful.
    • Nationality — start typing, the list filters.
    • Level — Beginner / Intermediate / Advanced (optional).
    • First registration — leave today's date, or backdate if needed.
    • Start date / Validity — defaults to today + 30 days. Pick 45/60/90/180 if it's a longer package.
    • Expiry — auto-calculated. You can override.
  4. Sport enrollments — pick the sport, coach, number of classes, and price. Click + Add sport for additional sports if the member enrolls in more than one.
  5. Payment method: cash or card.
  6. Click Save.
✓ What happens automatically:
  • An invoice is created (e.g. INV0001) summing all sports.
  • A subscription record is created for each sport.
  • The receipt opens in a new tab — you can print or save as PDF.

Renew a subscription

When a member's subscription is about to expire and they pay for another month:

  1. Find the member: Members → click on the row, OR History → click their name.
  2. Click the red + Add Renewal button.
  3. If the member has multiple sports, pick which one to renew (the form pre-fills with that sport's details).
  4. Adjust:
    • Classes — usually 8
    • Validity — 30/45/60/90/180 days
    • Amount paid — defaults to the previous price, override if needed
    • Start date — today by default
    • Expiry — auto-calculated
  5. Click Save Renewal.
✓ Result: A new subscription row appears in their history, a new invoice is created (e.g. INV0042), and the renewal counter for that sport bumps up by 1.

Add a sport to an existing member

If a member currently does MMA and wants to also start Kick Boxing:

  1. Open the member profile and click Edit.
  2. Scroll to Sport enrollments.
  3. Click + Add sport.
  4. Pick the new sport (Kick Boxing), coach, classes, and price.
  5. Click Save.
✓ Result: A new subscription row is created for the added sport AND a new invoice is generated for the amount.

Mark attendance (daily)

  1. Go to Attendance.
  2. Pick the month at the top, and which day(s) you want to see (or leave on "All").
  3. Each row is one (member × sport × coach). Click a cell under a day number:
    • First click → Y (green, present)
    • Second click → N (red, absent)
    • Third click → empty (not marked)
  4. Use the ↑ Today button (top right) to jump to today's column.
  5. Click ✓ Mark all present (today) to fast-mark everyone Y for today, then change a few to N for absentees.
💡 Filters: Use Coach / Sport / Student search to narrow the grid before marking — much faster than scrolling.

Per-student attendance PDF

The ⬇ PDF button on the right of each row prints that student's attendance card for the selected month — great for showing parents during pickup.

Class schedule

The 🗓 Schedule page (under Main in the sidebar) shows the full week of classes — what's running, when, with which coach. Perfect for answering customer questions like "When is the Kick Boxing class?" or "Who teaches Karate on Sunday?".

How the grid works

  • Rows = time slots from 3PM to 9PM (1-hour blocks)
  • Columns = days of the week — Saturday through Thursday (Friday is the off day)
  • Cells show one or more classes happening at that time. Each class shows the sport (with color + emoji) and the coach's name.

Editing the schedule (drag-and-drop)

  1. At the top of the page, you'll see colored sport tiles (Gymnastic, Boxing, MMA, etc.).
  2. Add a class: Drag a sport tile and drop it onto a cell → a pop-up asks which coach teaches it.
  3. Change the coach: Click any existing class block → the pop-up reopens.
  4. Move a class: Drag an existing class block from one cell to another.
  5. Remove a class: Click the × on the class block, or use "Remove class" in the edit pop-up.

Filters

Use the All coaches and All sports dropdowns to focus on one coach's classes or one sport's schedule. Non-matching classes dim out (still visible for context). The PNG export honors the active filter.

Export PNG (share on WhatsApp)

Click 📸 Export PNG to download a high-quality image of the schedule with the club branding. Perfect for sending to customers on WhatsApp or posting at the door.

✓ Tip: When the schedule changes (a coach swaps days, a new class is added), update it here once → the new PNG goes to customers. No need to redesign the printed poster.

Freeze a membership

If a member is traveling or injured and won't attend, freeze their subscription so days don't get wasted.

  1. Open the member's profile in History.
  2. Click ❄️ Freeze.
  3. Pick a duration: 1, 3, 7, 14, 30, 60, or 90 days — or type a custom number.
  4. Optionally write a reason (travel, injury, vacation).
  5. Click ❄️ Apply Freeze.
✓ Result: Their expiry date shifts forward by the same number of days. They're marked ❄️ Frozen on the dashboard and members list until the freeze ends.

To end a freeze early, use the ⏯ Unfreeze button on the profile.

Track trial members

The Trials page shows everyone who joined on a free trial. Use it to follow up before the trial ends so they convert to paying members.

Invoices

The Invoices page lists every invoice generated by the system.

Invoices are created automatically when you:

  • Register a new member with paid enrollment
  • Add a renewal via + Add Renewal
  • Edit a member and add a new sport
  • Complete a POS sale
  • Save a rental booking

Manual invoice

You can also create a one-off invoice via + New Invoice — useful for refunds or corrections.

Filters

Filter by month, category, activity, or search by customer name / invoice ref / description. Click any row to view details. Click 📄 PDF to print the receipt.

💡 Invoice references start at INV0001 and increment for each new invoice. The number never reuses or skips.

Product sales (POS)

For selling uniforms, gloves, drinks, etc.

Step 1 — Set up your products (one-time)

  1. Go to Products.
  2. Click + Add Product.
  3. Enter name, category, price, stock quantity, and low-stock threshold (e.g. alert at 5).
  4. Save.

Step 2 — Make a sale

  1. Go to Sales → click + New Sale.
  2. Pick a customer (existing member, walk-in by name, or anonymous).
  3. Add items: pick the product, set quantity. Cart updates totals.
  4. Apply discount if needed.
  5. Set payment method, amount paid (full or partial).
  6. Click Complete Sale.
✓ Automatic: Stock decreases, an invoice is created (category = Product), and a receipt opens for printing.
⚠️ Low stock alert: Products that drop below their threshold show a red badge on the Products page so you know to reorder.

Rental bookings

For renting the football court, boxing room, or swimming pool to outside customers.

New booking

  1. Go to Rentals → + New Booking.
  2. Pick the facility (rate auto-fills).
  3. Date, start time (optional), hours — total amount calculates automatically.
  4. Customer: Type the customer's name OR mobile number in the search box at the top. If they've booked before, click their name in the dropdown — name, mobile, and QID autofill. If they're new, just type the details in the fields below.
  5. Pick payment method, add notes (e.g. "Birthday party").
  6. Click 💾 Save & Create Invoice.
✓ Automatic: An invoice is created (category = Court Rental / Boxing Room) and the customer is saved for next time — search them by phone or name.

Customer history

Click 👥 Customers at the top of the Rentals page to see every rental customer, sorted by total revenue. Click 📜 next to any customer to see their full booking history.

Default hourly rates

Click ⚙ Rates on the Rentals page to set/change the default hourly rates per facility.

Log expenses

  1. Go to Expenses.
  2. Click + New Expense.
  3. Enter description (e.g. "May electricity bill"), amount, category, date, payment method (cash/card), and whether it's Monthly (recurring like rent) or Equipment (one-off purchase).
  4. Save.
💡 Categories: Equipment, Cleaning, Utilities, Marketing, Subscriptions, Transport, Operations, Rent, Coach Pool, Coach Commission, Salary. Pick whichever fits — used in Reports for breakdown.

Coach salaries & commission

The Salaries page shows each coach's monthly compensation.

  • Base salary — fixed monthly amount.
  • Commission — calculated as (coach's active-member revenue) × (coach's %). Editable per coach on the Coaches page.
  • Status — Paid / Pending. Click to toggle.
💡 Only active-member revenue counts toward commission. If a member expires or freezes, their past payment still appears in revenue but doesn't pay commission again.

Dashboard — what to look at first thing in the morning

  • Total Revenue card — combined for the last two months with the % trend arrow.
  • Active Members — includes a breakdown of frozen / completed / expired below.
  • Expiring this week — list of members whose subscription ends in the next 7 days. Tap to renew.
  • Revenue chart — current month vs previous month, revenue vs expenses vs profit.
  • Top coaches — quick ranking by students.

Reports

The Reports page is for deeper analysis. Pick a period (any month, year, or all-time) and you get:

  • Total revenue / expenses / profit for the period
  • Revenue split by category (Membership, Court Rental, Boxing Room, Product, Other)
  • Revenue split by sport
  • Top expenses for the period
  • Exportable summary

Coach Performance

Visual dashboard ranking your coaches by:

  • Commission earned (sorted bars)
  • Total revenue brought in
  • Number of students (column chart)
  • Attendance rate (% colored: green ≥75%, orange ≥40%, red below)

Use the month selector to compare different months. The metrics auto-update as members and invoices change.

Backup & Restore

⚠️ MOST IMPORTANT: All your data is stored in this browser only. If you receive a new version of the app, YOU MUST BACKUP FIRST, then restore after replacing files. Otherwise data could be lost.

📌 The right way to update the app to a new version

✓ Good news: Starting from v85, replacing the app folder with a newer version does NOT touch your data. Your members, invoices, attendance, schedule — everything stays. You only need to manually re-import data if you choose to (e.g. to load corrected Excel sheets).
  1. Save a backup as insurance — Dashboard → 💾 Backup now, OR sidebar → 💾 Quick backup.
  2. Close the browser tab.
  3. Replace the old app folder with the new version's folder on your computer.
  4. Open index.html from the new folder. Log in.
  5. Your data is automatically loaded. Continue working with the new features.
  6. If something looks wrong and you want to revert, go to Settings → 📂 Restore from backup → pick the JSON from step 1.

📥 The ONLY way data ever changes

Once you've started using the app:

  • You add/edit data manually (Add Member, Add Renewal, ring up a sale, log an expense, etc.)
  • You import Excel sheets via the Data Import page (replaces only the data type you import — e.g. importing Members.xlsx replaces members but not invoices)
  • You restore a JSON backup via Settings (replaces everything with the backup's contents)

App updates (replacing the folder) never touch your data. The bundled demo data (207 sample members) is ignored unless you explicitly click "🧪 Load demo data" in Settings.

🆕 First launch on a new computer

When you open the app for the very first time (or after a Hard Reset), you'll see a Welcome panel on the Dashboard with four options:

  • 📥 Import my Excel sheets — go to Data Import to load Members.xlsx, Attendance.xlsx, etc.
  • 📂 Restore from backup — load a previous JSON backup
  • ➕ Add members manually — start from zero, add one at a time
  • 🧪 Just load demo data to try it out — fills the app with 207 sample members so you can explore. Clear later when you're ready for real data.

Make a manual backup anytime

  1. Go to Settings → Data Management.
  2. Click 💾 Backup all data. A JSON file downloads — save it somewhere safe (Google Drive, external drive, email to yourself).

Restore from backup

  1. Go to Settings → Data Management → 📥 Restore from backup.
  2. Pick your backup file. The app replaces all current data with the backup.

Why JSON instead of Excel?

The Excel export (Members.xlsx, Attendance.xlsx, etc.) only covers what those four sheets contain. The JSON backup includes EVERYTHING — invoices, rentals, rental customers, products, sales line items, schedule, freezes, renewal counters per sport, settings, theme preference. Use Excel for sharing reports with non-technical people; use JSON for app-to-app data transfer.

Export to Excel

Go to Data Export — download Members, Attendance, Expenses, or Sales as Excel files individually, or all four as a single zip.

✓ Habit: Backup once a week minimum. Once a day during busy months. ALWAYS before installing a new version.

Click the theme icon in the top-right of the sidebar (🌙 / ☀️ / 📜 / 🎨) to cycle, or pick directly from Settings → Appearance:

  • ☀️ Light — bright clean white, for daytime use

  • 🎨 Colorful — purple gradient, fun and energetic

Tips & Tricks

  • Keyboard search: On most pages, the search box accepts partial matches across name, phone, sport, and coach.
  • Multi-sport members: One physical person can be enrolled in unlimited sports. Each gets its own subscription and counts separately in attendance.
  • Same family, same phone: Use the same phone number for siblings — search by phone shows them all together.
  • Expiring follow-up: Check the Expiring page every Monday — call/WhatsApp members expiring in the next 7 days.
  • Bulk attendance: Click ✓ Mark all present (today) first, then change just the absentees to N. Way faster than clicking each Y.
  • Customer history on rentals: Type a phone number in the rental form's search box and any past customers with that phone show up instantly.
  • Print receipts: Every invoice can be printed as a PDF (📄 button on its row). Use the browser's print dialog → Save as PDF, or send to printer.
  • Locking down: Always log out when leaving the computer (click your name at the bottom of the sidebar).

Troubleshooting

"My data disappeared!"

Check that you're on the same browser and computer where you logged the data. The CRM stores data locally per-browser. If you accidentally clicked "Clear browsing data" in Chrome settings, the data may be gone. This is why backups matter.

"The app shows old data after I updated"

Go to Settings → Hard reset → Reload latest seed data. This forces the app to load the fresh version. Backup first if you have custom data.

"Numbers on Dashboard look wrong"

Dashboard numbers include only invoices created in the system. If you imported old data without invoices, the dashboard won't reflect that. Use the Generate latest invoice button on the Invoices page to backfill missing invoices for members.

"Mobile-friendly?"

The app works on phones and tablets — the sidebar collapses to a hamburger menu. Best experience is on a laptop/desktop though, especially for the attendance grid.

"How do I export everything?"

Go to Data Export and click 📦 Download All. You get a zip with Members.xlsx, Attendance.xlsx, Expenses.xlsx, and Sales.xlsx — all in the original format the club used in Excel.

"Coach commission seems off"

Commission is calculated only on revenue from members who are currently Active. If a member expired and didn't renew, their past payment counts in total revenue but not in commission base. This protects against paying commission on churned customers.

"Anything else?"

Reach out to the developer who set this up. The app version is shown in the bottom-left of the sidebar (e.g. v4.39.0) — share that when reporting an issue.
